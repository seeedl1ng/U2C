### -В чём отличия этого форка от оригинала?
1. Исправления недочётов оригинала 
2. Немного бо́льшая производительность
3. Добавлен [Distant Horizons](https://modrinth.com/mod/distanthorizons)
4. Более частые обновления для клиентских модов
5. Вырезаны worldgen моды = меньшее время запуска
6. Добавлен [Default Options](https://modrinth.com/mod/default-options) (при обновлении форка, ваши настройки графики, клавиш, не будут сбрасываться)
7. Более новая версия LWJGL 3
8. Последняя версия Forge

Минимальные требования:
- Java 17 JRE
- 4.5ГБ ОЗУ (Выделенной для сборки)?
- Минимум 2ГБ свободного места на диске?

Оригинал: https://modrinth.com/modpack/chaotic_bp-release
